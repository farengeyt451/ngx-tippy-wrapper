## Recommended versions to use

- **Angular 14, 15, 16**: 6.x.x
- **Angular 13**: 5.x.x
- **Angular 12**: 4.x.x
- **Angular 9, 10, 11** 3.x.x
- **Angular 8**: 2.x.x

## Installation

Install from npm:

```js
npm i ngx-tippy-wrapper
```

Install from yarn:

```js
yarn add ngx-tippy-wrapper
```
